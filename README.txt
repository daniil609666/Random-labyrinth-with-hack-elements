Случайный лабиринт от DAN609


Случайный лабиринт от DAN609 с элементами взлома ПК
Для корректной работы игры нужны английская раскладка и права администратора!
Подробнее см. в файле Preview.png


Состояния игры (state)
Игра имеет несколько состояний, которые активируются при достижении конца лабиринта:

State 1 - Режим Paint:
Открывает Paint
Создает треугольник и кубик
Показывает сообщение "Hello! Hack3d!"
State 2 - Режим Calculator:
Открывает калькулятор
Выполняет вычисление 1+1
Показывает вопрос "Is 1+1=2?"
State 3 - Режим камеры:
Пытается сделать фотографию через веб-камеру
Сохраняет фото как "camera.png"
Показывает сообщение "I can see you!"
State 4 - Режим троллинга:
Случайное перемещение окон
Случайное движение курсора
Изменение позиций окон
State 5 - Режим системного доступа:
Имитация получения системного доступа
Открытие командной строки
Симуляция ввода команд
State 6 - Режим вируса:
Показывает сообщение об "удалении вируса"
Создает файл deleted.txt
Имитация сканирования системы
State 7 - Финальное состояние:
Блокирует ввод
Создает файл Congratulations.txt
Показывает финальное сообщение
Изменяет обои рабочего стола (если включена danger_functionality)
